Termlista

Termer från blogginlägget "FakeCaptcha" med korta förklaringar.

FakeCaptcha
En phishingmetod som lurar användaren att själv köra skadlig kod via falsk CAPTCHA.
Phishing
Angrepp där angripare lurar användare att göra något farligt eller lämna ut info.
CAPTCHA / reCAPTCHA
Test som ska skilja människor från botar, ofta missbrukat i bluffar.
PowerShell
Windows-kommandoskal och skriptspråk som ofta utnyttjas i attacker.
mshta.exe
Windows-program som kan köra HTML/skript och används ibland som "living-off-the-land".
Run (Win+R)
Windows‑dialog för att köra kommandon snabbt.
Clipboard (urklipp)
Temporärt minne för kopierad text, som kan fyllas med skadliga kommandon.
Malvertising
Skadlig annonsering som leder till bluff- eller malware-sidor.
SEO-poisoning
Manipulation av sökresultat för att få skadliga sidor att ranka högt.
Redirects
Omdirigeringar som skickar dig vidare till en annan, ofta skadlig, sida.
Fake tech support
Bluffsupport som får dig att ringa eller göra farliga instruktioner.
EDR
Endpoint Detection and Response, verktyg som upptäcker och stoppar attacker på klienter.
Sysadmin
Systemadministratör som driftar och skyddar IT‑miljöer.
LOLBins
"Living-off-the-land"‑verktyg i OS som missbrukas av angripare.
-EncodedCommand
PowerShell‑flagga för att köra kod i kodad form, vanligt i attacker.
-WindowStyle Hidden
PowerShell‑flagga som kör kommandon dolt utan fönster.
explorer.exe / powershell.exe
Windows‑processer som ofta ses i misstänkta kedjor.
Command line (cmdline)
Textbaserade kommandon som styr program och processer.
AV (antivirus)
Säkerhetsprogram som upptäcker och tar bort skadlig kod.
MFA
Multifaktorautentisering, extra steg ut över lösenord.
Domain (doman)
Namn på en webbplats, t.ex. example.com.
Punycode / IDN-homoglyfer
Tekniker som gör falska domäner svåra att skilja från riktiga.
HTTPS
Krypterad anslutning mellan webbläsare och server.
Certificate (certifikat)
Digitalt bevis som ligger bakom HTTPS‑låset i webbläsaren.
Safe Browsing
Tjänst som varnar för kända skadliga webbsidor.
Stealer
Malware som stjäl inloggningar, cookies eller annan data.
RAT
Remote Access Trojan, ger angriparen fjärrkontroll över datorn.
Crypto mining
Missbruk av din dator för att utvinna kryptovaluta.
Cloudflare CAPTCHA
CAPTCHA‑tjänst som ibland missbrukas på bluff‑sidor.
Web crawler
Automatiskt program som söker igenom webben, ofta blockerat av angripare.
Malware
Skadlig kod som infekterar system.
