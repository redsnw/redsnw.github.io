Termlista

Termer fran blogginlagget "FakeCaptcha" med korta forklaringar.

FakeCaptcha
En phishingmetod som lurar anvandaren att sjalv kora skadlig kod via falsk CAPTCHA.
Phishing
Angrepp dar angripare lurar anvandare att gora nagot farligt eller lamna ut info.
CAPTCHA / reCAPTCHA
Test som ska skilja manniskor fran botar, ofta missbrukat i bluffar.
PowerShell
Windows-kommandoskal och skriptsprak som ofta utnyttjas i attacker.
mshta.exe
Windows-program som kan kora HTML/skript och anvands ibland som "living-off-the-land".
Run (Win+R)
Windows dialog for att kora kommandon snabbt.
Clipboard (urklipp)
Temporart minne for kopierad text, som kan fyllas med skadliga kommandon.
Malvertising
Skadlig annonsering som leder till bluff- eller malware-sidor.
SEO-poisoning
Manipulation av sokresultat for att fa skadliga sidor att ranka hogt.
Redirects
Omdirigeringar som skickar dig vidare till en annan, ofta skadlig, sida.
Fake tech support
Bluffsupport som far dig att ringa eller gora farliga instruktioner.
EDR
Endpoint Detection and Response, verktyg som upptacker och stoppar attacker pa klienter.
Sysadmin
Systemadministrator som driftar och skyddar IT-miljoer.
LOLBins
"Living-off-the-land" verktyg i OS som missbrukas av angripare.
-EncodedCommand
PowerShell-flagga for att kora kod i kodad form, vanligt i attacker.
-WindowStyle Hidden
PowerShell-flagga som korer kommandon dolt utan fonster.
explorer.exe / powershell.exe
Windows-processer som ofta ses i misstankta kedjor.
Command line (cmdline)
Textbaserade kommandon som styr program och processer.
AV (antivirus)
Sakerhetsprogram som upptacker och tar bort skadlig kod.
MFA
Multifaktorautentisering, extra steg ut over losenord.
Domain (doman)
Namn pa en webbplats, t.ex. example.com.
Punycode / IDN-homoglyfer
Tekniker som gor falska domaner svarta att skilja fran riktiga.
HTTPS
Krypterad anslutning mellan webblasare och server.
Certificate (certifikat)
Digitalt bevis som ligger bakom HTTPS-laset i webblasaren.
Safe Browsing
Tjanst som varnar for kanda skadliga webbsidor.
Stealer
Malware som stal inloggningar, cookies eller annan data.
RAT
Remote Access Trojan, ger angriparen fjarrkontroll over datorn.
Crypto mining
Missbruk av din dator for att utvinna kryptovaluta.
Cloudflare CAPTCHA
CAPTCHA-tjanst som ibland missbrukas pa bluff-sidor.
Web crawler
Automatiskt program som soker igenom webben, ofta blockerat av angripare.
Malware
Skadlig kod som infekterar system.
